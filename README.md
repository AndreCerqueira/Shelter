# Shelter
## Game developed in Unity WEBGL, by André Cerqueira

## How to play
1. Open [Shelter](https://andrecerqueira.github.io/games/Shelter/index.html)

## Commands

- A & D -> Walk

- Espaço -> Jump

- E -> Transform

- F -> Pick items

- Segurar click esquerdo -> Aim snow ball

- Largar click esquerdo -> Throw snow ball
